# ROULETTE
Mobile Roulette variable calculator made with Ionic Capacitor using Angular

# Steps to run this project:

1. Run :
```bash
npm i
```
2. Finally Run : 
```bash
ionic serve
```

# Steps to build this project:

1. Run :
```bash
npm i
```
2. Finally Run : 
```bash
ionic capacitor build android
```
